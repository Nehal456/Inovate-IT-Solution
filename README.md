# Innovate IT Solution

## Overview
Innovate IT Solution is a dynamic IT service platform built using Flask and deployed on AWS cloud infrastructure. The project focuses on scalable deployment architecture, responsive frontend design, and backend integration.

---

## Features
- Responsive frontend UI
- Flask backend architecture
- Contact management system
- SQLAlchemy database integration
- AWS cloud deployment
- Gunicorn + NGINX configuration
- Git/GitHub workflow integration
- - PostgreSQL database integration

---

## Tech Stack
- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLAlchemy
- AWS Elastic Beanstalk
- Gunicorn
- NGINX
- GitHub
- PostgreSQL

---

## Deployment Architecture

User → NGINX → Gunicorn → Flask Application → Database

---

## Installation

```bash
git clone <repo-url>
cd innovate-it-solution
pip install -r requirements.txt
python app.py
```
